Detailed Description

creates the field variables related to the PU kinetics

This file firstly creates an IOdictionary labeled as kineticsProperties. Two different kinetics models have been introduced. The first one generic follows the work of Baser and Khakhar by solving PDEs for the conversion of polyol and isocyanate. However, the second model, RF-1, applies a surrogate model derived from Predici results. Additionally, all the required fields including the sources have been created in this file
