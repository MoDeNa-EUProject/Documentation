Detailed Description

Class for defining 'forward mapping' models.

A forward mapping model can be thought of as a constitutive equation, or a scaling function. 

                   f(X) => Y = 2 * X
            +---------------------+
            |                     |
          |-+---|             |---v------|
                X                        Y
